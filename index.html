This is a description of the various parts of the database. It tells people who are new to the database what each column means, and how to easily navigate the database to find what they want. In the future, this will also hold the search feature for the database, allowing individuals to search names, descriptions, and commanders. You can also filter lists to ones that contain colors, a strategy, or if a list has a primer.
